# project-agents

Canonical **per-repo** agent bodies (max 3). Emitted to harness `agents/` on `omakase learn`.

Edit these files, then re-run `omakase learn` (or `--project-agents-only`) to refresh native stubs.

Namespaced ids: `omakase-<package>-<role>`. Core leads (@omakase-engineer, etc.) come from the package — never replace them here.
