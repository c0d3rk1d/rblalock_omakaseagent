# Omakase

**The chef's standard.**

This is the same Omakase skill available at:

- **Recommended install:** `npx omakase skills install`
- **This repo:** https://github.com/rblalock/omakaseagent

## When to Use

Use Omakase when you want senior-level output with no AI slop. It enforces:

- The 12 Omakase Rules
- The 8-bullet Critique Rubric (mandatory on all significant work)
- Ruthless simplicity
- Explicit "Why this approach" reasoning
- Persistent project taste memory (`.omakaseagent/`)

## Commands

- `/omakase init` — Bootstrap `.omakaseagent/` with taste + decisions
- `/omakase engineer` — Full senior Engineering persona
- `/omakase critique` — Domain-aware critique with proper merge behavior
- `/omakase plan` — Senior planning with explicit reasoning
- Or just describe what you want in plain language

The skill detects intent and applies the right standards automatically.
